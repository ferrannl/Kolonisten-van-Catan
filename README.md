Dit project is een Java-implementatie rond het bordspel **Kolonisten van Catan**, ontwikkeld als groepsopdracht (*Groep I*) voor het vak **Informatica 1**.

**EN**

This repository contains a Java project based on the board game **Settlers of Catan**, created as a group assignment (*Group I*) for the **Informatics 1** course.
